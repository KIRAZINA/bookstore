# Bookstore API

     Spring Boot приложение для онлайн-книжного магазина.

     ## Функции
     - JWT аутентификация (login/register).
     - CRUD для книг (admin only).
     - Корзина и заказы.
     - H2 in-memory DB.

     ## Запуск
    mvnw spring-boot:run


## API
- POST /api/auth/login
- GET /api/books

## Технологии
- Spring Boot 3.x
- Spring Security + JWT
- Hibernate
- Maven
